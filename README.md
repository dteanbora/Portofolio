#################### design ui 1 = Kopi Senja – UI/UX Design (Restaurant Website)

Kopi Senja merupakan proyek perancangan antarmuka (UI/UX) untuk website restoran yang berfokus pada penyajian menu, suasana, dan informasi kontak secara visual dan interaktif.
Proyek ini dibuat menggunakan Figma sebagai media perancangan, dengan tujuan menghasilkan desain yang modern, estetis, dan mudah digunakan oleh pengguna.

## Project Overview

Desain ini mencakup beberapa halaman utama yang umumnya terdapat pada website restoran, yaitu:
- Home Page
- Menu Page
- Contact Page

Setiap halaman dirancang dengan mempertimbangkan konsistensi visual, hierarki informasi, dan kenyamanan pengguna.

## Design Objectives

- Meningkatkan pengalaman pengguna melalui layout yang jelas dan terstruktur
- Menyediakan tampilan visual yang konsisten dan menarik
- Mempermudah pengguna dalam menemukan informasi penting
- Menghadirkan nuansa yang sesuai dengan konsep restoran
